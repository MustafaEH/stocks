# Stocks Tradeflow

Monorepo for a stocks trading demo app — includes a Python backend and a React/Vite frontend.

Quick start (Docker Compose):

1. Start services:

```bash
docker-compose up --build
```

2. Backend API: http://localhost:8000 (FastAPI)
3. Frontend: http://localhost:5173

Development (backend):

- See `backend/trading-main/README.md` for backend details and requirements.

Development (frontend):

- See `frontend/tradeflow-dashboard-main/README.md` for frontend dev commands.
